Plugin documentation

Goals available for this plugin:

Goal Description
hk2:compile Compiles application sources
hk2:hk2-compile Replacement of the default javac mojo that uses APT, so that we can not only compile source code but also generate HK2 related metadata at the same time.
hk2:createscripts A Rudimentary Mojo to create scripts for Windows and non-Windows Platforms, based on an intuitive file in a form similar to registry file.
hk2:run Launches the module system by using all the dependencies as modules.

Not specifying "@execute phase=compile" so that it runs quickly even with a large project.
hk2:package Creates a jar with a special manifest entry.

Mostly copied from JarMojo. See http://jira.codehaus.org/browse/MNG-2789 why we can't rely on it.
hk2:hk2-test-compile No description.
hk2:analyse-packages Analyses bundle manifest entries in a repository and generates various statistics like: no. of bundles, no. of exported packages, package wiring details and split-packages.
hk2:compute-osgi-version Converts the project version into the OSGi format and set that to "project.osgi.version" property.
hk2:analyse-dependency Does static analysis of classes in a module to ensure that module dependency is correctly set up.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK 1.5
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You could run 'mvn -up' to get the latest version of this plugin, or specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.sun.enterprise<groupId>
          <artifactId>hk2-maven-plugin<artifactId>
          <version>0.3.37-SNAPSHOT<version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.sun.enterprise<groupId>
        <artifactId>hk2-maven-plugin<artifactId>
        <version>0.3.37-SNAPSHOT<version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
